<h1>SAMURAI-BOT</h1>
Cách chạy: 

`py main.py`

Game control: <br>
[← / →]: Di chuyển trái / phải. <br>
[↑]: Nhảy. <br>
[Spacebar]: Chém. <br>
[x]: Ném shuriken.<br>
[c]: Dùng item làm chậm thời gian. <br>

Nhiệm vụ:
Tiêu diệt "Nhện Máy". <br>
